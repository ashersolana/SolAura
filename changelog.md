# Changelog ✦

All notable changes to SolAura will be documented here.

---

## [0.1.0] - 2026-03-13

### Added
- Initial project setup
- React + Tailwind CSS foundation
- Solana wallet adapter integration
- Natural language chat interface
- AI intent parser for wallet commands
- Real-time SOL balance fetching
- Token portfolio viewer
- Transaction history reader
- Security layer — private keys never touch AI
- Landing page with animated terminal demo
- Open source MIT license
- Contributing guidelines
- GitHub Pages deployment

### Security
- Bounded permissions implemented
- Prompt injection protection added
- User confirmation required for all transactions

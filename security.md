# Security Policy ✦

## Supported Versions

| Version | Supported |
| ------- | ------------------ |
| 0.1.x | ✅ Yes |

## Reporting a Vulnerability

At SolAura we take security seriously. Since SolAura handles Solana wallet interactions, we have strict security standards.

## 🔒 Our Security Principles

- Private keys **never** touch the AI layer
- All wallet actions require **explicit user confirmation**
- Bounded permissions prevent prompt injection attacks
- LLM only reads **public wallet data**
- Every transaction shows a confirmation screen before signing

## 🐛 Found a Bug?

If you discover a security vulnerability please **do not** open a public issue.

Instead contact us privately:

- **X:** [@EliteCryptoLord](https://x.com/EliteCryptoLord)
- **GitHub:** [@ashersolana](https://github.com/ashersolana)

## ⚡ Response Time

We aim to respond to security reports within **48 hours**.

## 🙏 Responsible Disclosure

We appreciate responsible disclosure and will credit researchers who report valid vulnerabilities.

---

*SolAura is open source and community driven. Security is our top priority when handling wallet interactions.* ✦
